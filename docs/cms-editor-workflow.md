# CMS redaktorflyt (Simone)

1. Gå til `https://dev.sound-of-simone-dev.pages.dev/admin/#/` og logg inn med GitHub.
2. Åpne `Nettsideinnhold -> Hovedinnhold`.
3. Gjør ønskede endringer i tekst og/eller bilder.
4. Trykk `Save` for å lagre utkast.
5. Gå til `Workflow` og send endringen videre til review.
6. Etter godkjenning publiseres endringen.

Notater:
- Bilder lastes opp til `public/images` via CMS.
- Publisering er satt til `editorial_workflow` (review-flyt), ikke direkte publish.
- Prelaunch: CMS er DEV-only. Produksjonsadmin skal ikke brukes før launch-cutover.
